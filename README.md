# Trabajo Práctico: Decision Transformer para Recomendaciones

Repositorio para el TP de Offline RL / Decision Transformer sobre los datasets Netflix o Goodreads.

## Requisitos previos
- Python 3.10+ recomendado.
- `git` y `bash`/terminal.

## Instalación del entorno
1) Crear entorno virtual:
```bash
python3 -m venv .venv
```
2) Activar el entorno:
```bash
source .venv/bin/activate  # Linux/macOS
```
3) Instalar dependencias:
```bash
pip install --upgrade pip
pip install -r requirements.txt
```

## Ejecutar notebooks en VS Code
- Abre el `.ipynb` deseado (ej. `notebooks/prueba.ipynb`).
- Selecciona el kernel `tp-dec-transformer` o el intérprete `.venv/bin/python`.
- Ejecuta las celdas normalmente.
